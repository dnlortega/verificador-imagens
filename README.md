# PixelArmor PRO

**PixelArmor** é uma ferramenta ultrarrápida, ágil e local para detecção de imagens corrompidas e análise forense de metadados truncados. Construída com um visual elegante, *glassmorphism* e alta concorrência.

## 🚀 Tecnologias Principais

- **Next.js 16** (App Router)
- **Tailwind CSS 4**
- **Shadcn UI** (Componentes Acessíveis e Estilizados)
- **Web Workers Ready** (Processamento Cliente-Lado)

## 💻 Como Rodar Localmente

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.

---

> Desenvolvido com foco absoluto em UI/UX Premium e Performance. Nenhuma imagem é enviada para o servidor; tudo ocorre 100% no seu navegador.
